== Description ==

在编辑器里输入微信公众号文章链接，点击采集微信文章就自动抓取到编辑器里,非常方便用户获取微信文章内容.


## 功能概述

* 支持编辑器旁边显示 输入微信文章链接
* 支持block块编辑器和经典编辑器一键抓取微信文章
* 支持一键采集微信公众号文章
* 支持一键采集微信公众号文章的图文
* 支持采集文章图片自动本地化转换
